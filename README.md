## Features

This extension parses and analyzes thread dump and generates a table to display useful information.

![OUTPUT](/images/output.png)


## How To Use

* Mac: cmd+shift+p
search "Analyze Thread Dump" command

* Windows: ctrl+shift+p
search "Analyze Thread Dump" command
